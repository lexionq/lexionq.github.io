Anka Browser v2.0 Yayınlandı! #

Daha önceki yazımda Anka Browser’ı C++ diline geçirdiğimi ve C++ kullandığım ilk versiyonumu yazdığımdan bahsetmiştim. Şimdi ise Anka Browser’ın C++ ile yazdığım ikinci versiyonu olan V2.0 yayınlandı. FAKAT ANKA BROWSER’I TAMAMEN TEKRAR TANITACAĞIM.

Anka Browser Nedir? #

Anka Browser, Qt kütüphanesi ile tasarlanan, motor olarak QWebEngine kullanan, QWebEngine kullanmasından kaynaklı Chromium tabanlı yerli bir tarayıcıdır. İçerisinde bir çok arama motoru bulundurur. Anka Browser’da kullanabileceğiniz arama motorları şunlardır:

  • Google
  • DuckDuckGo
  • Brave Search
  • Artado
  • Bing
  • Startpage

Bu arama motorları arasından dilediğinizi seçip kullanabilirsiniz.

Yapabileceğiniz Özelleştirmeler ve Ayarlar: #

Anka Browser’da şu ayarları ve kişiselleştirmeleri yapabilirsiniz:

  • Sekme Rengi Değiştirme
  • Geçmişinizi Silme
  • Yer İmlerinizi Silme
  • Tarayıcı Yazı Tipini Değiştirme(Henüz eklenmedi v2.2 veya v3.0’da eklenecek.)

Anka Browser Nasıl İndirilir? #

Şuan için sadece Linux’ta indirme desteği var. Fakat çok yakında Windows için de indirme desteği eklenecek. Peki ya Linux’ta nasıl indirilir:

1. GitHub Adresine Gidin #

Projemizin GitHub adresine buraya tıklayarak gidebilirsiniz. Ardından ekranın sağ tarafında göreceğiniz Releases kısmından en son (Latest) olarak görünen linke tıklayın ve o “Release"i indirin.

2. Kurulum #

İndirdiğiniz dosyayı eğer gerekirse sudo chmod +x /indirme_yolunuz_/Anka ile çalıştırılabilir hale getirin. Projeyi .deb olarak paketleyektim fakat yetişmedi. Şimdilik gerekli olan kütüphaneleri kurmanız lazım: QtCore, QtWidgets, QtWebEngineWidgets. Eğer Debian/Ubuntu veya tabanlı bir dağıtım kullanıyorsanız: sudo apt install qt6-base-dev qt6-base-dev-tools qt6-webengine-dev bu şekilde indirebilirsiniz. Çalıştırılabilir dosya sizin için derlenmiş olduğundan dolayı sadece kütüphaneleri yüklemeniz yeterli. Anka’yı istediğiniz gibi kullanabilirsiniz.

Şimdilik anlatacaklarım bu kadar. Bir sonraki versiyonu yine buradan duyuracağım. İyi günler dilerim.
